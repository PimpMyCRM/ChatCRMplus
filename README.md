## Features

### Direct & Group Chats
Start private 1-to-1 conversations or create named group chats for teams, departments, or projects.

### Real-Time Delivery
Messages appear instantly with a lightweight real-time experience, without requiring a heavy infrastructure.

### Typing Indicators
Live typing feedback makes conversations feel immediate and natural.

### Read Receipts
Know when messages have been seen by each participant.

### Multi-Language Support
Supports 6 languages out of the box:

- English
- Spanish
- Portuguese
- Italian
- French
- German

Language is detected automatically from the user's EspoCRM profile settings.

### Presence Status
Users can be shown as:

- Online
- Away
- Offline

### Profile Avatars
User photos are pulled directly from EspoCRM for instant recognition.

### Persistent Chat History
Conversations remain available inside the CRM for ongoing team visibility and continuity.

### Mobile Responsive
Works across desktop and mobile screen sizes.

### Native EspoCRM Feel
Designed to feel like part of EspoCRM, not like a bolted-on third-party widget.
